# SMART-ATM-SYSTEM
In this Project I haveCreated Sign In Page For New Account Holder In Bank. That Contains the Information of Account Holder like Personal Details,PAN number, Aadhaar number etc. With the Help of Java Swing And Applet.  Database Connection and Table are created with the help of MySQL.
